House Price Prediction

This project is based on estimating house prices in Bengaluru using features such as the Number of BHKs (Bedroom, Hall, and Kitchen), Number of Bathrooms, Location of the House, etc. This project includes Exploratory data analysis, Data preprocessing, Feature Engineering, Outlier Detection, Data Visualisation, and Training the model.
